# Number Guessing Game 🎯

A command-line number guessing game built in Python using the random module.

## Features
- Generates a random number between 1-100
- Gives hints (higher/lower) based on user's guess
- Tracks and displays total number of guesses taken

## How to Run
python guessing_game.py

## Concepts Used
- Random module
- While loops
- Conditional statements (if/elif/else)
- User input handling
- Counter variables
